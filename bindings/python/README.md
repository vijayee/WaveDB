# WaveDB Python Bindings

Python bindings for [WaveDB](https://github.com/vijayee/WaveDB) - a Hierarchical
B+Trie database with MUMPS-style multi-dimensional subscript access.

This package is a work in progress. See the project root for full documentation.

## Status

Experimental - API may change. This package provides FFI-based bindings to the
native `libwavedb` shared library.
